Title Screen Modifications.
<h2 align="center">Features</h2>
[Screen] Multiple custom text with customizable size, max width, line height, align and position.<br><br>
[Menu] Multiple external links with customizable text.<br><br>
[Menu] Customizable width (relative), visible rows, align and position (relative).<br><br>
[Menu] Quit option with customizable text (NW.js only).<br><br>
Multilanguage support (English, Spanish, Portuguese, Italian, French, German, Russian, Japanese, Korean and Chinese).<br>
